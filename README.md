# Convo
A visual novel made with Renpy for Resolution-gamedev week2
- Check it out on itch: https://callerer.itch.io/convo

## About
- It follows the main character from the previous visual novel Qava (Check it out here: https://callerer.itch.io/qava )

  Lin interacts with a new customer who seems familiar to him. He seems to have a cool pendant, ask him abt it?
- It's a short sequel to the main story.
